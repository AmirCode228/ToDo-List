# ToDo List
**ToDo List** на **Python** с **сохранением** данных в файл **JSON**

## Описание
**ToDo List** на **Python** с **сохранением** данных в файл **JSON**,  без **сторонних** библиотек
Есть **русская версия** rus/main.py и **английская версия** eng/main.py
Данные **сохраняются** в **data.json**

## Особенности
* Данные **сохраняются** в **data.json**
* Есть **русская** и **английская** версия
* **Без сторонних библиотек**

## Как использовать
1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/AmirCode228/ToDo-List.git
   ```
2.
  * Для **русской** версии перейдите в папку **rus**:
    ```bash
    cd ToDo-List\rus
    ```
  * Для **английской** версии перейдите в папку **eng**:
    ```bash
    cd ToDo-List\eng
    ```
3. Запустите файл:
   ```bash
   python main.py
   ```
